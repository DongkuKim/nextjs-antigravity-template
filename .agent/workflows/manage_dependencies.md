# Manage Dependencies

1.  **Initialize Project**
    -   If starting fresh: `npm init -y`

2.  **Add Dependencies**
    -   To add a package: `npm install <package_name>`
    -   Example: `npm install react`
    -   For dev dependencies: `npm install -D <package_name>`

3.  **Remove Dependencies**
    -   To remove a package: `npm uninstall <package_name>`

4.  **Sync Environment**
    -   To install dependencies from `package.json`:
    -   Command: `npm install`
